Bootstrap Components!

Bootstrap comes with numerous built in components. From nav bars, to forms and even buttons. Almost everything you may need to quickly build and test your prototypes. Here you can find some examples of common components you may find yourself using:

Bootstrap Colors!

Bootstrap comes with a built-in set of pre-defined colors. The prefix of the class determines the color target, and the end of the class name determines the color:

The “text” or .text-(color) class is used to change the color of the text: class="text-primary"

Creates blue text

The “bg” or .bg-(color) class is used to change the color of the background: class="text-primary"

Creates blue background

Bootstrap Buttons

Bootstrap has custom button styles that can be used in combination with bootstrap colors. The class "btn" can also be used on a tags or input elements to create button links.

There are 2 main types of buttons. There is a normal filled button:

button class="btn btn-info"


But there is also the outline button:

button class="btn btn-outline-info"

Buttons have pre built hover effects included in bootstrap. Buttons can also be different colors depending on which color you decide to use.
